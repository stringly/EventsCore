System.Int32 userId

An integer Id of the User associated with the registration
System.String userName

A string containing the User's display name.
System.String email

A string containing the User's email address.
System.String contact

A string containing the User's primary contact number.
Remarks

The Registration Entity exists as a part of the EventRegistrations aggregate root. Registrations can only be created/update/removed from the aggregate root. Registrations will be created with the default status of "Pending."

Exceptions
Type Condition
EventRegistrationAggregateArgumentException

Thrown when:

  • The provided userId parameter is 0 or out of range.
  • The provided userName parameter is empty/whitespace.
  • The provided email parameter is empty/whitespace.
