public class EventDates : ValueObject
Remarks

This object contains the Start/End Dates for an Event, as well as the Registration Period Start/End Dates. These are encapsulated in this object so that the dates can be validated.

Parameters
Type Name Description
System.DateTime evStart

DateTime object representing the Date/Time the Event is to begin. Must be before End Date.
System.DateTime evEnd

DateTime object representing the Date/Time the Event is to end. Must be after Start Date.
System.DateTime rgStart

DateTime object representing the Date/Time the Registration Period for the Event is to begin. Must be before Event Start Date.
System.DateTime rgEnd

DateTime object representing the Date/Time the Registration Period for the Event is to end. Must be before Event Start Date and after Registration Start Date.
IDateTime dateTimeProvider

An implemementation of IDateTime used to access the system time.
Remarks

The constructor enforces the rule that an Event cannot have a Start Date for any date in the past. This check is performed against the System time. Keep this requirement in mind for testing.

Exceptions
Type Condition
EventDatesInvalidException

Thrown when:

  • The evStart parameter is in the past.
  • The evStart parameter is a date after the evEnd parameter in the past.
  • The rgStart parameter is a date after the evStart parameter.
  • The rgStart parameter is a date after the rgEnd parameter.
